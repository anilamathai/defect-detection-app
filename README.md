# Defect Detection System

This is a real-time defect detection app built with **YOLOv8** and **Streamlit**.  
It can detect cracks, holes, lines, and other defects in ceramic tiles.

## Features

- Upload an image and detect defects
- Use your laptop webcam for real-time detection
- Powered by a custom-trained YOLOv8 model

## Installation

Clone the repo and install requirements:

```bash
git clone https://github.com/YOUR_USERNAME/defect-detection-app.git
cd defect-detection-app
pip install -r requirements.txt
```
